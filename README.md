# 🖤 GOTIKA — Gothic Fashion E-Commerce Shop

Интернет-магазин стильной, концептуальной готической одежды и аксессуаров **GOTIKA**. Проект представляет собой современное фронтенд-приложение с интерактивным каталогом, детальными карточками товаров, возможностью фильтрации по категориям и динамической корзиной покупок.

---

## ✨ Особенности проекта

- **Тематическая тёмная эстетика**: Уникальный UI/UX дизайн, вдохновленный готической культурой, с использованием глубоких темных тонов и изящных акцентов.
- **Интерактивный каталог товаров**: Динамическое отображение ассортимента (плащи, корсеты, обувь, аксессуары) на основе структурированной базы данных.
- **Интерактивная корзина**: Добавление товаров в один клик, изменение количества, удаление элементов и мгновенный перерасчет итоговой стоимости заказа.
- **Адаптивный интерфейс (Responsive Design)**: Сайт полностью оптимизирован под мобильные устройства, планшеты и десктопные экраны.

🚀 Инструкция по локальному запуску
Для запуска проекта на вашем компьютере выполните следующие шаги:

Клонируйте репозиторий:

Bash
git clone [https://github.com/ВАШ_НИКНЕЙМ/gotika-fashion-shop.git](https://github.com/solynarany/gotika-fashion-shop.git)
Перейдите в папку проекта:

Bash
cd gotika-fashion-shop
Распакуйте папки images и modules
Запустите проект:

Если вы используете Visual Studio Code, установите расширение Live Server, нажмите правой кнопкой мыши на файл index.html и выберите "Open with Live Server".

Либо просто откройте файл index.html в любом современном браузере (Chrome, Firefox, Edge, Safari).
